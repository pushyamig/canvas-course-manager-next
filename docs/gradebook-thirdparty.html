Format Third-Party Gradebook

This feature allows you to Format a CSV file exported from an external tool for importing into the Canvas Gradebook. This tool creates a new version of the uploaded file so it only includes students in the selected sections and is formatted appropriately for uploading to Canvas gradebook using the Canvas gradebook import function..

Who Can Use Format Third-Party Gradebook?
  • Root Account Administrator
  • Sub-Account Administrator
  • Support Consultant
  • Teachers and other course managers who normally have access to the People tool
