Add U-M Users

Bulk adding U-M users to course sections with Course Manager is an administrator feature that offers some benefits over the People tool method:

  • Add users from a CSV file
  • Create a section on the fly and add users to it
  • Add users to multiple existing sections at one time
  • Bypasses the built-in Canvas People tool email notifications sent to new course members
Who Can Add U-M users?

Bulk adding U-M users to sections is limited to Canvas administrator roles:

  • Root Account Administrator
  • Sub-Account Administrator
  • Support Consultant
  • Teachers and other course managers can request additional assistance by contacting 4help@umich.edu.

Tip: Instructors and other roles with People tool access can add U-M users in bulk by entering a list of email addresses or Uniqunames (login ID) in the People tool, being aware that the users may receive a new course invitation message from Canvas.

How to Add U-M Users to a Single Section?

This feature allows administrators to select an existing section or create a new section to accept the users.

  • Click the Add U-M users button in Canvas Course Manager.
  • Choose the “1 Section at a Time” option and click Select.
  • Download the example CSV file, if needed.
  • Prepare and save your CSV file following the onscreen instructions for header and row content.
  • Drag and drop, or click the link to upload your saved CSV file.
  • CSV file data is presented back to you. Verify the results and click Submit.
How to Add U-M Users to Multiple Sections?

This feature allows administrators to add U-M users to more than one existing section at the same time. New sections cannot be created with this tool.

To use this feature you must include the Canvas section ID (not the section SIS ID) in your CSV file. A list of course section ID is included with this tool to help with CSV file preparation.

  • Click the Add U-M users button in Canvas Course Manager.
  • Choose the “Using Multiple Sections” option and click Select.
  • Download the example CSV file, if needed.
  • Prepare and save your CSV file following the onscreen instructions for header and row content. A list of course section IDs is presented on screen and a CSV version is available for download.
  • Drag and drop, or click the link to upload your saved CSV file.
  • CSV file data is presented back to you. Verify the results and click Submit.
